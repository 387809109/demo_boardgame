## Religious Struggle Procedures (Summary)

> Full rules: RULEBOOK_FOR_DEVELOPMENT.md / RULEBOOK_SECTION_NORMALIZED_ZH.md §18
> Reference card image: his_ref/img/classified/religious_struggle.jpg

### Theological Debate (§18.5)

**Attacker Dice**: debate value + 3 (attacker bonus +1 built in; Eck/Gardiner: additional +1; Papal Inquisition: +2)

**Defender Dice**: debate value + 2 (if uncommitted) or debate value + 1 (if already committed)

**Hits**: each roll of 5 or 6 = 1 hit

**Result**: Side with more hits wins; flip (hit difference) spaces to winner's religious influence. Tied → random 2nd round (tied again = no result). Winner may burn/disgrace losing debater if hit difference > defender's debate value.

### Reformation Attempt (§18.3)

**Protestant Dice** (min 1):
- +1 per adjacent Protestant space
- +1 per adjacent Protestant reformer
- +1 per adjacent Protestant stack
- +2 if reformer in target space
- +2 if Protestant stack in target space
- +1 if applicable debater bonus
- +1 if Printing Press in play
- +1 if Luther's 95 Theses attempt

**Papacy Dice** (min 1):
- +1 per adjacent Catholic space
- +1 per adjacent Jesuit university
- +1 per adjacent Catholic stack
- +2 if Jesuit university in target space
- +2 if Catholic stack in target space

**Result**: Compare highest single die. Protestant wins ties in target language zone; Papacy wins ties elsewhere. Winner flips space. Bible translation adds +1 to each Protestant die.

### Counter-Reformation Attempt (§18.4)

**Papacy Dice** (min 1):
- +1 per adjacent Catholic space
- +1 per adjacent Jesuit university
- +1 per adjacent Catholic stack
- +2 if Jesuit university in target space
- +2 if Catholic stack in target space
- +1 if applicable debater bonus

**Protestant Dice** (min 1):
- +1 per adjacent Protestant space
- +1 per adjacent Protestant reformer
- +1 per adjacent Protestant stack
- +2 if reformer in target space
- +2 if Protestant stack in target space

**Result**: Compare highest single die. Papacy wins ties if Paul III is Pope in target language zone. Augsburg Confession: -1 to each Papacy die. Winner flips space.
